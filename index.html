HTML5 REFERENCE v.0

<Tags>

The HTML5 element tags(<> </>) are listed below. Those that do not require a closing tag(</>) are marked with a * symbol.

Document Description
<!DOCTYPE HTML>* The <!DOCTYPE html> declaration represents the document type, and helps browsers to display web pages correctly.
<!--comments-->* HTML comments are not displayed in the browser, but they can help document your HTML source code.
<html> The HTML document itself begins with <html> and ends with </html>


Section Description
<body> The <body> element contains all the contents of an HTML document, such as headings, paragraphs, images, hyperlinks, tables, lists, etc.
<nav> The <nav> tag defines a set of navigation links.
<header> The <header> element represents a container for introductory content or a set of navigational links.
<article> The <article> tag specifies independent, self-contained content.


Grouping Description
<p> The <p> element defines a paragraph.
<ul> The <ul> tag defines an unordered (bulleted) list.
<ol> The <ol> tag defines an ordered list. An ordered list can be numerical or alphabetical.
<li> The <li> tag defines a list item.The <li> tag is used inside ordered lists(<ol>), unordered lists (<ul>), and in menu lists (<menu>).


Embedding Description
<video> The <video> tag is used to embed video content in a document.
<img>* The <img> tag is used to embed an image in an HTML page.
<audio> The <audio> tag is used to embed sound content in a document


Form Description
<button> The <button> tag defines a clickable button.
<input>* The <input> specifies an input field where the user can enter data.
<label>* The <label> element represents a caption for an item in a user interface.


Meta Data Description
<base>* The <base> tag specifies a default URL and target for all links on a page.
<head> The <head> element is a container for metadata (data about data)
<title> The <title> tag defines the title of the document
<link>* The <link> tag defines the relationship between the current document and an external resource.
